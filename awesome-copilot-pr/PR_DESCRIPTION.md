## Add 11 GTM (Go-to-Market) Skills for Technical Product Founders

### What this adds

11 agent skills covering go-to-market strategy for AI and technical product founders. These are non-engineering skills — frameworks for positioning, pricing, enterprise sales, partnerships, and board communication.

### Skills included

| Skill | Description |
|-------|-------------|
| `gtm-positioning-strategy` | Find and own a defensible market position |
| `gtm-technical-product-pricing` | Pricing strategy for technical products (usage-based, freemium, enterprise) |
| `gtm-ai-gtm` | Go-to-market for AI products (trust gaps, copilot/agent framing, variable-cost pricing) |
| `gtm-enterprise-account-planning` | Strategic account planning with MEDDICC, mutual action plans |
| `gtm-enterprise-onboarding` | Four-phase onboarding from contract to value realization |
| `gtm-product-led-growth` | Self-serve acquisition, activation, freemium conversion |
| `gtm-partnership-architecture` | Build and scale partner ecosystems |
| `gtm-developer-ecosystem` | Developer-led adoption and ecosystem programs |
| `gtm-0-to-1-launch` | Launch new products from idea to first customers |
| `gtm-operating-cadence` | Meeting rhythms, metric reporting, quarterly planning |
| `gtm-board-and-investor-communication` | Board decks, investor updates, executive communication |

### Why this belongs in awesome-copilot

Most skills in this repo are engineering-focused. Founders and GTM operators ask their AI agents positioning, pricing, and sales questions every day and get generic answers. These skills teach methodology — not theory — from real GTM motions at hypergrowth companies.
