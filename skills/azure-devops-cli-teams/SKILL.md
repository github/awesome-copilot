## When to Use This Skill
- You want to manage teams in an Azure DevOps project.
- You need to automate team management tasks or integrate with scripts and CI/CD workflows.


## Prerequisites

**CLI Version:** 2.81.0

```bash
# Install Azure DevOps extension
az extension add --name azure-devops
```

Use the reference commands in [REFERENCE.md](references/REFERENCE.md) for specific team management tasks.
