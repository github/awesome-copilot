## When to Use This Skill

- You want to manage Azure Repos, including repositories, pull requests, refs, and policies.
- You need to automate repository management tasks or integrate with scripts and CI/CD workflows.

## Prerequisites

**CLI Version:** 2.81.0 (current as of 2025)

```bash
# Install Azure DevOps extension
az extension add --name azure-devops
```
