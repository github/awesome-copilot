# Emdash Skills

Autonomous product-building OS that turns one-line prompts into deployed products. 14 skill categories cover the full lifecycle from architecture through deploy and post-launch growth.

Source: [megabytespace/claude-skills](https://github.com/megabytespace/claude-skills)

## When to Use

- Building a new SaaS, landing page, or web app from scratch
- Deploying full-stack applications to Cloudflare Workers
- Setting up complete CI/CD, testing, and observability pipelines
- Architecting with Angular, Hono, D1, KV, R2, Clerk, Stripe
- Running multi-agent parallel workflows for large projects
- Applying comprehensive SEO, accessibility, and performance standards
