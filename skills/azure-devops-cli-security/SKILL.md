## When to Use This Skill

- You want to manage Azure DevOps security groups and permissions.
- You need to automate security management tasks or integrate with scripts and CI/CD workflows.


## Prerequisites

**CLI Version:** 2.81.0 (current as of 2025)

```bash
# Install Azure DevOps extension
az extension add --name azure-devops
```

## Guidelines
Use the reference commands in [REFERENCE.md](references/REFERENCE.md) for specific security group and permission management tasks.
