## When to Use This Skill
- You want to manage Azure Pipelines, including runs, builds, releases, variables, and agent infrastructure.
- You need to automate pipeline management tasks or integrate with scripts and CI/CD workflows.

## Prerequisites

**CLI Version:** 2.81.0 (current as of 2025)

```bash
# Install Azure DevOps extension
az extension add --name azure-devops
```
