## When to Use This Skill
- You want to manage users in an Azure DevOps organization.
- You need to automate user management tasks or integrate with scripts and CI/CD workflows.

## Prerequisites

**CLI Version:** 2.81.0 (current as of 2025)

```bash
# Install Azure DevOps extension
az extension add --name azure-devops
```

## Guidelines
Use the reference commands in [REFERENCE.md](references/REFERENCE.md) for specific user management tasks.
