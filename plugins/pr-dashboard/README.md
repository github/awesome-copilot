# PR Dashboard Plugin

Open a GitHub PR dashboard in the browser for a given date range and role filter.

## Requirements

- [GitHub CLI (`gh`)](https://cli.github.com/) installed and authenticated (`gh auth login`)
- Node.js

## Skills

### pr-dashboard

Generates and opens a self-contained HTML dashboard showing your pull requests, including status, review state, and CI status.

**Usage examples:**
- "show my PRs"
- "open PR dashboard last 2 weeks"
- "PR dashboard this month reviews"
- "show all PRs last 30 days"

## Installation

```bash
copilot skill install pr-dashboard@awesome-copilot
```
