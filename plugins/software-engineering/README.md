# Software Engineering Plugin

A collection of specialist AI agents covering the full software engineering lifecycle — architecture, DevOps, security, technical writing, and documentation quality. All agents are cloud-agnostic, language-agnostic, and framework-agnostic.

## Agents

| Agent | Description |
|---|---|
| `adr-generator` | Structured Architecture Decision Records with coded consequences and quality checklist |
| `cloud-architect` | Senior Cloud Architect producing Mermaid diagrams and NFR analysis — no code generation |
| `devops-expert` | DevOps infinity loop specialist (Plan → Code → Build → Test → Release → Deploy → Operate → Monitor) |
| `markdown-accessibility-assistant` | Improves markdown accessibility using GitHub's 5 best practices + markdownlint |
| `project-architecture-planner` | Holistic architecture planner with tech-stack evaluation, cost modelling, and phased scalability roadmap |
| `repo-architect` | Scaffolds and validates `.github/` and `.opencode/` agentic project structures |
| `se-devops-ci` | GitOps & CI specialist — deployment debugging, pipeline design, reliability patterns |
| `se-security` | Security reviewer — OWASP Top 10, Zero Trust, OWASP LLM Top 10 |
| `se-architect` | System architecture reviewer — Well-Architected frameworks, scalability, AI-specific concerns |
| `se-tech-writer` | Technical writing specialist — docs, blogs, tutorials, ADRs, user guides |
| `technical-content-evaluator` | Elite curriculum quality reviewer — grading, documentation wrapper detection, exercise validation |

## Usage

Install this plugin to make all 11 agents available in VS Code Copilot. Select the appropriate agent from the agent picker for your current task.
