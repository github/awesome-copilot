# Azure Pricing Plugin

Fetches real-time Azure retail pricing using the Azure Retail Prices API. Use when the user asks about Azure service costs, SKU comparisons, or billing estimates.

## Installation

```bash
# Using Copilot CLI
copilot plugin install azure-pricing@awesome-copilot
```

## What's Included

### Commands (Slash Commands)

| Command | Description |
|---------|-------------|
| `/azure-pricing:azure-pricing` | Fetch real-time Azure retail pricing for any service, region, or SKU using the Azure Retail Prices API |
