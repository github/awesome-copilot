# RAG Azure Builder Plugin

Opinionated Azure RAG starter plugin with guided onboarding, indexing strategy, and grounded QA setup patterns.

## Installation

```bash
copilot plugin install rag-azure-builder@awesome-copilot
```

## What's Included

### Agents

| Agent | Description |
|---|---|
| `rag-onboarding` | End-to-end onboarding flow for Azure RAG with architecture, cost, and readiness checks. |

### Skills

| Skill | Description |
|---|---|
| `rag-azure-setup` | Plan Azure resource baseline and security defaults for RAG. |
| `rag-indexer` | Define ingestion, chunking, metadata, and retrieval strategy for Azure AI Search. |
| `rag-qa-engine` | Shape grounded Q and A behavior, citations, and quality checks. |
